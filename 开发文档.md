# WordBook 单词本开发文档（Claude Code 专用版）

## 项目简介

WordBook 是一个个人单词记忆工具。

基于：

* Vue 3
* Vite
* Vue Router
* Dexie.js
* Tailwind CSS
* PWA

构建。

目标：

* 手机浏览器直接使用
* 支持添加到桌面
* 本地离线运行
* 不依赖服务器
* 数据完全存储在浏览器 IndexedDB
* 支持间隔复习算法
* 支持故事阅读辅助记忆

---

# 核心原则

## 原则1：业务数据与代码完全分离

禁止将单词、故事内容直接写入 Vue 组件。

所有业务数据均存放于独立文件：

```txt
src/data/
├── words.json
├── stories.json
└── app-config.json
```

组件只负责读取和展示。

---

## 原则2：优先保证可维护性

允许后续通过替换 JSON 文件扩展：

* 单词数量
* 故事数量
* 配置项

无需修改业务代码。

# 数据文件规范

## words.json

格式：

```json
[
  {
    "id": 1,
    "word": "come",
    "pos": "v.",
    "meaning": "来",
    "phonetic": "/kʌm/"
  }
]
```

字段说明：

| 字段     | 说明     |
| -------- | -------- |
| id       | 唯一ID   |
| word     | 英文单词 |
| pos      | 词性     |
| meaning  | 中文释义 |
| phonetic | 音标     |

注意：

* word必须小写
* id必须唯一
* phonetic允许为空字符串

---

## stories.json

格式：

```json
[
  {
    "id": 1,
    "title": "A Small Adventure",
    "content": "Tom walks to the river...",
    "translation": "汤姆走向河边..."
  }
]
```

字段说明：

| 字段        | 说明     |
| ----------- | -------- |
| id          | 唯一ID   |
| title       | 标题     |
| content     | 英文正文 |
| translation | 中文译文 |

注意：

* content保留完整英文
* translation保留完整中文

---

## app-config.json

格式：

```json
{
  "dailyReviewCount": 20,
  "dailyNewWordCount": 10,
  "maxFuzzySearchResults": 3
}
```

字段说明：

| 字段                  | 说明               |
| --------------------- | ------------------ |
| dailyReviewCount      | 默认复习数量       |
| dailyNewWordCount     | 默认新学数量       |
| maxFuzzySearchResults | 模糊匹配最大结果数 |

# Dexie 数据库设计

数据库名称：

```js
VocabDB
```

Schema：

```js
db.version(1).stores({
  words: 'id, word',
  reviews: '++id, &wordId, nextReviewAt, interval, ease, repetitions'
});
```

说明：

```txt
&wordId
表示唯一索引
一个单词只允许存在一条复习记录
```

# 今日任务生成规则

配置来源：

```txt
app-config.json
```

用户修改后：

```txt
localStorage优先
配置文件兜底
```

生成规则：

### 第一步

获取所有：

```txt
nextReviewAt <= 当前时间
```

的到期单词。

---

### 第二步

按：

```txt
nextReviewAt 升序
```

排序。

---

### 第三步

取：

```txt
dailyReviewCount
```

个复习单词。

---

### 第四步

如果数量不足：

从没有 review 记录的单词中补充。

数量：

```txt
dailyNewWordCount
```

---

### 最终任务数

```txt
复习任务 + 新学任务
```

---

# 单词查询规则

点击故事中的单词时：

执行以下步骤：

### Step1

转小写

```txt
Running
↓
running
```

---

### Step2

移除标点

```txt
running,
↓
running
```

---

### Step3

精确匹配

```txt
running
```

---

### Step4

简单词形还原

支持：

```txt
running → run

studies → study

walked → walk

goes → go
```

---

### Step5

模糊匹配

按：

```txt
includes
Levenshtein
```

任选一种实现。

最多返回：

```txt
3条
```

结果来源：

```txt
maxFuzzySearchResults
```

---

### Step6

仍未找到

显示：

```txt
词库未收录
```
